> **Multiideal** é uma plataforma de e-commerce de elite e hub empreendedor, combinando a densidade de interface do Mercado Livre com a curadoria de alta performance da Shopee e Amazon.

## 🚀 Por que este projeto?

Este projeto foi desenvolvido para demonstrar como a tecnologia pode servir à sustentabilidade sem sacrificar a **melhor experiência de usuário (UX)**. O mercado de impacto verde muitas vezes sofre com designs simples demais; o EcoShop quebra esse padrão trazendo uma **High-Density UI** inspirada em líderes do setor.

### 🌟 Diferenciais Técnicos
- **Performance Mobile-First**: Interface ultra-otimizada para dispositivos móveis com grades de 2 colunas e navegação por gestos (swipe carousels).
- **Checkout Inteligente**: Integração com API de localização e fluxo de pagamento Pix via Mercado Pago.
- **Design de Alta Conversão**: Uso de selos dinâmicos (FULL, Frete Grátis) e estimativas de entrega em tempo real para maximizar o e-commerce funnel.
- **Arquitetura Modular**: Componentes reutilizáveis e tipagem rigorosa com TypeScript para escalabilidade empresarial.

## 🛠️ Tech Stack

- **Frontend**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Componentes UI**: [shadcn/ui](https://ui.shadcn.com/)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Gerenciamento de Estado**: Context API (Cart, Auth, Location)
- **Forms**: React Hook Form + Zod

## ✨ Principais Funcionalidades

- 📁 **Navegação Semântica**: Rodapé e Header organizados para uma jornada de usuário fluida.
- 📱 **Mobile Header**: Barra de busca integrada ao topo e localização persistente.
- 💳 **Fluxo de Checkout**: Simulação completa desde a identificação até o QR Code do Pix.
- 🏆 **Eco-Membro Gold**: Área de benefícios premium inspirada no sistema Meli+.
- 📊 **Impact Insight**: Páginas institucionais que comunicam o valor e a missão da marca.

## 🏗️ Como Rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/usuario/ecoshop.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
